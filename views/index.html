Image Search Abstraction Layer

A solution to the Image Search Abstraction Layer challenge.

Features

  • Get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • Paginate through the responses by adding a ?offset=2 parameter to the URL.
  • Get a list of the most recently submitted search strings.

Example Usage

Search 

        curl "https://jy-fcc-image-search.glitch.me/api/images/pugs?offset=20" | jq
        {
          "query": "pugs",
          "offset": 20,
          "count": 10,
          "items": [
            {
              "image_url": "http://www.pugnow.com/wp-content/uploads/2016/04/fly-pug-300x300.jpg",
              "alt_text": "Flying Pugs | Extreme Pugs | Pug can Fly",
              "page_url": "http://www.pugnow.com/flying-pugs/"
            },
            {
              "image_url": "https://www.pets4homes.co.uk/images/breeds/13/large/3918021b2f92bd036598a095fb7e45de.jpg",
              "alt_text": "Pug - Alchetron, The Free Social Encyclopedia",
              "page_url": "https://alchetron.com/Pug"
            },
            ....
          ]
        }
