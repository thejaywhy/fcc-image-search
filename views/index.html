Image Search Abstraction Layer

A solution to the Image Search Abstraction Layer challenge.

Features

  • Get the image URLs, alt text and page urls for a set of images relating to a given search string.
  • Paginate through the responses by adding a ?offset=2 parameter to the URL.
  • Get a list of the most recently submitted search strings.

Example Usage

Search via FORM 

        curl -d "term=pug" https://jy-fcc-image-search.glitch.me/api/images
        {"url":"tbd","text":"tbd","source":"tbd url"}

Search via JSON 

        curl -d '{"term": "pug"}' -H "Content-Type: application/json" https://jy-fcc-image-search.glitch.me/api/images
        {"url":"tbd","text":"tbd","source":"tbd url"}

Get Recent Searches 

        curl -v https://jy-fcc-image-search.glitch.me/api/latest/images
        {[{"term":"tbd","when":"sometime"}]}
